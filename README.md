# 臺大獎學金備註助手

一個簡單的 Chrome 擴充功能，讓你在瀏覽 [臺大獎學金頁面](https://advisory.ntu.edu.tw/CMS/Scholarship) 時，可以：

- 新增「備註」欄
- 輸入個人筆記（如：v看過了、OK要申請）
- 關閉瀏覽器後內容不消失（自動儲存）
- 點擊備註欄不會誤觸跳轉

## 安裝方式

1. git clone 此專案
2. 進入 Chrome 擴充功能頁面：`chrome://extensions/`
3. 開啟「開發人員模式」
4. 點選「載入已解壓縮的擴充功能」，選擇專案資料夾
